# Hướng dẫn chạy dự án Web Quản lý Cửa hàng Cắt tóc

## Cài đặt

1. Di chuyển vào thư mục dự án:
```bash
cd fontend/fontend
```

2. Cài đặt dependencies:
```bash
npm install
```

## Chạy dự án

```bash
npm run dev
```

Dự án sẽ chạy tại: http://localhost:5173

## Cấu trúc trang

### 1. Trang chủ (/)
- Giới thiệu về cửa hàng
- Hiển thị các dịch vụ
- Thông tin liên hệ
- Nút đăng nhập/đặt lịch

### 2. Trang User (/user)
- Đặt lịch cắt tóc
- Xem lịch sử đặt lịch
- Quản lý thông tin cá nhân

### 3. Trang Admin (/admin)
- Dashboard với thống kê
- Quản lý đặt lịch
- Quản lý dịch vụ
- Quản lý nhân viên
- Quản lý khách hàng

## Tính năng chính

- Giao diện responsive, thân thiện
- Điều hướng giữa các trang
- Form đặt lịch cho khách hàng
- Bảng quản lý dữ liệu cho admin
- Màu sắc chủ đạo: Đen (#1a1a1a) và Vàng gold (#d4af37)
